# Exercício — Estruturas de dados em C#

### Crie um programa que:
1. Crie diferentes estruturas de dados  
2. Armazene elas em uma coleção   
3. Percorra essa coleção  
4. Imprima:
    - valor
    - tipo
    - tamanho (se aplicável)

### Estruturas que você deve usar
- `int`
- `string`
- `double`
- `bool`
- `List<int>`
- `Dictionary<string, int>`
- `Tuple`
- `HashSet<int>`
- `int[] (array)`
